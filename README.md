# Grocery store demo app

## Overview

Simple app to present groceries

### Features:

-   fetch and present groceries
-   filtering products 
-   adding products to basket
-   present basket 
-   change currency

### Stack:

-   Swift
-   MVVM + Coordinators
-   DI + Factories
